Privacy Policy

Aegis Moderation Bot

This Privacy Policy explains how Aegis Moderation Bot ("Aegis") collects, uses, and handles data when operating within Discord servers.

1. Information We Collect

Aegis may process and store the following information to function correctly:

  • Discord User IDs
  • Discord Guild (Server) IDs
  • Channel IDs and Role IDs (for configuration and ignored lists)
  • Moderation actions (warnings, timeouts, removals, etc.)
  • Warning records and moderation history
  • Server-specific configuration settings

2. Message Content

Aegis may temporarily process message content to detect spam, abuse, or policy violations. Message content is not permanently stored unless it is included as part of a moderation action (such as a reason provided by a moderator).

3. How Data is Used

Data collected by Aegis is used solely to:

  • Provide moderation functionality
  • Track warnings and moderation history
  • Detect spam and abusive behaviour
  • Store server-specific settings and configurations

4. Data Storage

Data that is collected, is stored securely using local storage systems (such as SQLite databases). This data is not sold, shared, or distributed to third parties.

5. Data Retention

Data is retained only as long as necessary for the bot to function properly. Server owners may request data removal where applicable.

6. Data Sharing

Aegis does not share, sell, or distribute user or server data to any third parties.

7. User Rights

Users and server owners may request information about stored data or request its removal, where technically possible.

8. Changes to this Policy

This Privacy Policy may be updated at any time. Continued use of Aegis after changes indicates acceptance of the updated policy.

9. Contact

If you have any questions regarding this Privacy Policy, you may contact me directly at: aegisbot.support@gmail.com
