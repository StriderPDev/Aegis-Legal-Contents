What is Aegis?

Aegis is a moderation assistant designed to help server staff manage and protect their community. It provides structured tools, clear moderation tracking, and automated protection against spam and abuse.

Key Features

  • Anti-spam detection (fast spam, repeat spam, mention spam)
  • Configurable moderation system per server
  • Warnings and moderation history tracking
  • Interactive control panel (/modpanel)
  • Security role system for staff hierarchy
  • Ignored channels and role support

How to Use Aegis

Aegis is designed to be simple and structured. After adding the bot to your server:

  • Use /help to view guidance and navigation
  • Use /aegisconfig to review your server configuration
  • Use /modpanel to control moderation features
  • Assign staff roles using /setsecuritylevel
  • Test features in a safe channel before full use
